Ollama Chat Desktop App

Beschreibung:
Eine Python Desktop-Anwendung mit grafischer Oberfläche, die sich mit der lokalen Ollama API verbindet und KI-Antworten generiert.

Technologien:
- Python
- ttkbootstrap
- Requests
- Ollama (Llama3)

Funktionen:
- KI-Chat über lokale Ollama API
- Eingabe und Anzeige von Antworten
- Fehlerbehandlung bei Verbindungsproblemen

Installation:

Pakete installieren:

pip install ttkbootstrap requests


Ollama starten:

ollama serve


Programm starten:

python OllamaVerbindung.py
